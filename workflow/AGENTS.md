# Workflow 目录 Agent 指南

## 目录用途

`workflow/` 存放可复用的工作流模板，用于自动化开发流程。

## 目录结构

```
workflow/
├── auto-dev-loop/       # 自动开发循环工作流
└── canvas-dev/          # Canvas白板驱动开发工作流
```

## 操作规范

### 允许
- 新增工作流模板
- 修改现有工作流配置
- 添加工作流文档说明

### 禁止
- 删除现有工作流（除非明确要求）

## 工作流规范

每个工作流应包含：
- `README.md` - 使用说明
- 配置文件或脚本
- 示例/模板文件（可选）
